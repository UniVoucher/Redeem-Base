Redeem Base

Redeem UniVoucher crypto gift cards without connecting a wallet or paying gas fees

This is an open-source web app. Free to clone and use.
Built using Cursor IDE, Claude 4 Sonnet, and UniVoucher MCP.

You can use it with the UniVoucher Partner Program to earn 1% partner fees from your users.

Developers: You can redeem cards without gas using our Redeem Base API (0% fees apply).
